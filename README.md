## Instalação Front-End
execute o comando abaixo.
 ```sh
npm install
```

## Instalação Back-End
navegue até a pasta /back-end.
 ```
cd back-end
```

execute o comando abaixo.
 ```sh
npm install
```
## Executando a Aplicação 

Após realizar a instalação, execute o comando abaixo
<strong>repita o processo na pasta /back-end</strong>.

```sh
npm start 
```
